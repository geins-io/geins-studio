# Features

This is a list of the main features available in Geins Studio to this date.

| Feature                    | Description                                                       | Available&nbsp;since |
| -------------------------- | ----------------------------------------------------------------- | -------------------- |
| **Quotations**             | Create, edit, send, and manage quotation lifecycle                | v0.3.0               |
| **Communications**         | Threaded messages (customer & internal) on quotations             | v0.3.0               |
| **Status transitions**     | Multi-step status workflows with confirmation dialogs             | v0.3.0               |
| **Dynamic page titles**    | Browser tab titles derived from breadcrumb trail                  | v0.3.0               |
| **Improved error states**  | Inline error states with retry for list pages and tables          | v0.3.0               |
| **Profile management**     | Manage your own user profile                                      | v0.1.4               |
| **Price lists**            | Create and manage price lists                                     | v0.1.0               |
| **Companies**              | Create and manage companies                                       | v0.1.0               |
| **Product selector**       | Advanced product selector                                         | v0.0.1               |
| **Authentication**         | Authenticate using Geins Auth                                     | v0.0.1               |
